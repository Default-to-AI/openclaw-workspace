# conventions

## Tasks
- Source of truth: `02_action/tasks/10_next-actions.md`
- Every task must include exactly one context tag: `#@deep-work #@computer #@phone #@errands #@calls #@home #@money #@content`
- Keep "Today" <= 3 items.

## Shopping
- Items start in `03_shopping/00_inbox.md`
- Move to `03_shopping/10_to-research/` as a dedicated note per item.
- Only move to `03_shopping/30_to-buy.md` after a shortlist + decision.
