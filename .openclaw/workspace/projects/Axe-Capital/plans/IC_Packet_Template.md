# Axe Capital — IC Packet Template (Public/Private Markets)

> Goal: enable a high-quality decision with a clear record of assumptions, risks, sizing logic, and monitoring plan.

## 0) Decision header
- **Proposal**: Initiate / Add / Reduce / Exit / Hedge
- **Instrument(s)**: ticker/ISIN, structure, venue
- **Direction**: long/short
- **Requested size**: % NAV, $ notional, expected gross/net impact
- **Time horizon**: catalyst-driven / multi-quarter / trading
- **Owner**: analyst / PM
- **Vote needed?** yes/no (define quorum and threshold)

## 1) One-page summary
- **Thesis** (1 paragraph)
- **Variant perception** (what the market is missing)
- **Catalysts** (timed list)
- **Base / bull / bear** outcome table (returns + probabilities)
- **Key risks** (top 3)
- **Exit plan** (what invalidates thesis)

## 2) Business / asset overview
- What it is, how it makes money (or payoff structure)
- Industry structure / competitors / positioning

## 3) Financials / valuation
- Key drivers and model outputs
- Valuation framework (multiples, DCF, comps, sum-of-parts, carry)
- Sensitivities (2–3 charts/tables)

## 4) Technicals and positioning
- Liquidity profile (ADV, borrow if short, options depth)
- Ownership / crowdedness proxies
- Seasonality or event calendar impacts

## 5) Risk section (must be explicit)
- **Downside analysis**: worst-case path, permanent vs mark-to-market
- **Factor exposures**: rates, credit, commodities, FX, style
- **Scenario / stress tests**: 2008-style, COVID shock, rates spike, etc.
- **Concentration**: single-name/sector/factor contributions
- **Kill criteria**: hard stops (price/time/thesis)

## 6) Sizing and implementation
- Position sizing rationale (Kelly-lite, risk budget, max loss)
- Entry plan (levels, tranches), hedge plan
- Execution plan (venues, timing, limits, slippage assumptions)

## 7) Compliance and operational readiness
- MNPI statement, restricted list check
- Settlement/custody considerations
- Documentation and counterparty considerations

## 8) Monitoring plan
- KPI dashboard for the thesis
- What to watch weekly/monthly
- Next catalyst review date

## 9) Appendices
- Model, primary sources, call notes, transcripts, exhibits

## 10) Decision log (filled after)
- Decision: approve/deny/defer
- Conditions: size cap, hedge required, time-box, data to collect
- Review date: ___
